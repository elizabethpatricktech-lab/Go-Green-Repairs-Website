# Go Green Repairs Website

A modern full-stack business website built for **Go Green Repairs**, an HVAC, Commercial Kitchen, and Bakery repair company.

The project is being rebuilt from a static HTML/CSS/JavaScript website into a scalable full-stack application using React and Django.

---

## Features

### Current Features

- Modern responsive React frontend
- Business information pages
- Services page
- Financing information
- Reviews page
- Contact form
- Image and video showcase
- Django Admin Panel
- Service and Review database models

### Planned Features

- User authentication
- Customer dashboard
- Service request system
- Service status tracking
- Customer review system
- Admin customer management
- Cloudinary media storage
- Email notifications
- Google Reviews integration

---
